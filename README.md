# RM Tools 1.4.0

Клиентский мод для персонала RaidMine на **Minecraft 1.21.11 / Fabric / LabyMod 4**.

## Основные возможности

- ЛКМ по автору сообщения в чате открывает меню наказаний. Упомянутые в тексте игроки не становятся кликабельными.
- Поддержка нестандартного масштаба и вертикального смещения SmartChat.
- Сетевой учёт ручных команд `/warn`, `/mute`, `/ban`, `/hub` и `/v`, включая команды, отправленные через SmartChat.
- Дневное время онлайна хранится между режимами и сбрасывается в **00:00 по Москве**.
- После `/hub` счётчик сразу замораживается. На игровых режимах он продолжает отсчёт с сохранённого значения.
- После выполнения дневной нормы время становится зелёным, а справа появляется панель «Уведомления и подсказки».
- Опциональный **AFK Kick** заранее отправляет модератора в `/hub` после 4:30–4:50 бездействия.
- Верхняя панель показывает баны, муты, варны, дневное время и состояние ваниша.
- Фиолетовый глаз означает включённый `/v`, тёмно-серый — выключенный.
- Right Shift открывает редактор HUD с анимацией, перемещением стрелками и масштабированием с восьми сторон. Ctrl меняет размер относительно центра.
- Настройки содержат готовые цветовые схемы, выбор шрифта, дневную норму, AFK Kick и отдельный файл запрещённых слов.
- После входа показывается анимированная заставка «С возвращением».
- После наказания автоматически сохраняется скриншот с ником и пунктом правил в имени файла.

## Горячие клавиши

- **ЛКМ по автору сообщения** — меню наказаний.
- **`/rmp Ник`** — резервное открытие меню.
- **Right Shift** — редактор HUD.
- В редакторе: стрелки — перемещение, Shift+стрелки — ускоренное перемещение, Right Shift или `C` — центрирование, `R` — сброс, Esc — сохранить.
- Перетягивание маркеров — изменение размера; удерживание Ctrl — симметрично от центра.

## Файлы конфигурации

- `config/rm-tools.json` — основные настройки и аккаунты персонала.
- `config/rm-tools/forbidden-words.json` — переносимый список запрещённых слов.
- `config/rm-tools/daily-state.json` — дневное время и статистика наказаний.

## Сборка

Требуется Java 21. GitHub Actions настроен в `.github/workflows/main.yml`.

```bash
gradle --no-daemon clean build
```

Готовый файл:

```text
build/libs/rm-tools-1.4.0.jar
```
